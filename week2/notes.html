Notes

  • Callbacks work like a domino effect
  • Callback example:
    function getInput(id){
    if (document.getElementById(id).value !== NaN){
    return parseFloat(document.getElementById(id).value);
    } else {
    return 0;
    }
    }
    function divideInHalf(number){
    return number / 2;
    }
    function doubleNumber(number){
    return number * 2;
    }
    function writeResult(value){
    return document.getElementById('anwer').innerHTML = value;
    }
    function doMathFunction(statement){
    const result = statement(getInput('input1'));
    writeResult(result);
    }
    Example OnClick:
    Halfing button on click is set to doMathFunction(divideInHalf)
    Double button is set to doMathFunction(doubleNumber)
